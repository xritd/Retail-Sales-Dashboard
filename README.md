# Retail Sales Dashboard

Simple end-to-end data analysis project for a fictional retail store (200 orders) across 2023-2024.

## Tools Used
- Python (pandas) – data cleaning & summaries  
- SQLite – database & SQL queries  
- Power BI – interactive dashboard  

## Key Insights
- North region generated the highest revenue  
- Webcams and GPUs are the top-selling products by quantity  
- Clear monthly sales trend visible

## Files
- `data/` – raw and cleaned CSV  
- `python/` – cleaning script  
- `sql/` – queries  
- `powerbi/` – .pbix file + screenshot
